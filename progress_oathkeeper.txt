  - Added strict command availability handling with `ErrCommandUnavailable`

- Files changed

- Learnings for future iterations:
  - Patterns discovered
    - Current `br` in this environment supports `--labels` for create/list filtering and not `--tag`; runtime fallback to label flags keeps compatibility while still requiring `br`.

    - `br close` emits info logs to stderr on success; parsing must rely on stdout for JSON data commands.
  - Useful context

    - Required verification command `pytest pkg/beads/beads_test.go -v` fails in principle because pytest does not collect Go test files.
